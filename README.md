# Мой сайт-визитка

Персональный сайт-визитка, созданный с использованием Flask.

## Технологии
- Python
- Flask
- HTML/CSS
- Font Awesome

## Установка и запуск
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите приложение: `python app.py`
3. Откройте http://localhost:5000 в браузере
